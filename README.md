# what is common_java?
A common utilities that can be used for a variety Java project.

# where to get this lib?
[maven](https://mvnrepository.com/artifact/ch.weetech/common-java)


# quick build
```
$ export JAVA_HOME=/usr/lib/jvm/jdk-11.0.5/
$ ./gradlew clean build
```
